# Portafolio Web

Sitio web de portafolio desarrollado con Astro, React, TypeScript y Tailwind CSS.

## Requisitos

- Node.js
- npm

## Instalacion

```bash
npm install
```

## Comandos

```bash
npm run dev
```

Inicia el servidor de desarrollo.

```bash
npm run build
```

Verifica el proyecto y genera la version de produccion.

```bash
npm run preview
```

Previsualiza la version generada.

## Estructura

- `src/pages`: paginas del sitio.
- `src/components`: secciones reutilizables del portafolio.
- `src/layouts`: layouts base.
- `src/hooks`: hooks personalizados.

## Personalizacion

El contenido principal se puede ajustar desde los componentes en `src/components`.
